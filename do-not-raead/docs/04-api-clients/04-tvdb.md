# TVDb Client

## Features

- Movies: Basic support
- TV Series: Full support
- Episodes: Full support

## Configuration

```yaml
tvdb:
  api_key: "your_key"
```

## Extras

Supported:
- Credits (cast only, no crew)
- Images (posters/backdrops)
- External IDs
- Content ratings

Not supported:
- Videos
- Keywords
- Watch providers

## Authentication

JWT token managed automatically.
