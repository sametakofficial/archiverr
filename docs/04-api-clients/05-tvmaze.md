# TVMaze Client

## Features

- Movies: Not supported
- TV Series: Full support
- Episodes: Full support

## Configuration

No API key required:

```yaml
tv_priority:
  - tvmaze
```

## Extras

Supported:
- Credits (cast/crew)
- Images
- External IDs (IMDB, TVDB, TVRage)

Not supported:
- Videos
- Keywords
- Content ratings
- Watch providers

## Rate Limit

No rate limit.
