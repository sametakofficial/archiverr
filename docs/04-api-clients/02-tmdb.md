# TMDb Client

## Features

- Movies: Full support
- TV Series: Full support
- Episodes: Full support
- All extras available

## Configuration

```yaml
tmdb:
  api_key: "your_key"
  lang: "en-US"
  region: "US"
```

## Extras

All extras supported:
- Credits (cast/crew)
- Images (posters/backdrops)
- Videos (trailers)
- Keywords
- External IDs
- Content ratings
- Watch providers

## Rate Limit

40 requests per 10 seconds (managed automatically)
